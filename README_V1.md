# Learn <insert-subject-here> by Doing

A compilation of the best hands-on, project-based resources for learning things that aren't programming — music, philosophy, science, art, languages, and more.

> *"Tell me and I forget. Teach me and I remember. Involve me and I learn." — Benjamin Franklin*

The best way to learn anything is to *do* it. Not watch a lecture, not read a textbook — but to sit down and struggle through it yourself. These resources all share that philosophy.

## How to Use This List

1. **Pick ONE subject** that excites you right now
2. **Start with a project or method book**, not a passive course
3. **Set a daily practice minimum** (even 15 minutes counts)
4. **Track your progress** — journal, record, photograph your work
5. **Share what you make** — feedback accelerates learning
6. **Come back here** when you're ready for the next level

# Music Learning Resources 🎸

A curated compilation of the best resources for learning music — starting with **guitar**.

> *Inspired by [build-your-own-x](https://github.com/codecrafters-io/build-your-own-x), but for learning to play music instead of building software.*

Resources are sorted by **media type** and tagged by **cost** so you can find exactly what fits your budget and learning style:

- `FREE` — completely free, no paywall
- `FREEMIUM` — solid free tier, paid options for more
- `PAID` — requires a subscription or one-time purchase (free trials noted where available)

## Websites & Structured Courses

| Resource | Cost | Notes |
|---|---|---|
| [JustinGuitar](https://www.justinguitar.com/) | `FREEMIUM` | The gold standard for free, structured beginner-to-intermediate guitar lessons. Run by Justin Sandercoe. Most content free; optional paid songs/app. |
| [Guitar Tricks](https://www.guitartricks.com/) | `PAID` | 11,000+ lessons, huge song library. $19.99/mo or $179.99/yr. 14-day free trial. |
| [TrueFire](https://truefire.com/) | `PAID` | 40,000+ lessons, 30,000 tabs, 20,000 jam tracks. Great for intermediate/advanced. $29/mo, $249/yr, or $2,499 lifetime. 30-day trial. |
| [JamPlay](https://jamplay.com/) | `PAID` | Genre-based learning paths, jam tracks. Partners with TrueFire for 1,500+ courses. Affordable tiers. |
| [Fender Play](https://www.fender.com/play) | `PAID` | Beginner-friendly guided paths from Fender. $19.99/mo or $149.99/yr. 7-day trial. |
| [JTC Guitar](https://jtcguitar.com/) | `PAID` | Masterclasses, jam tracks, interactive tabs. ~£19.99/mo. 14-day trial. |
| [FaChords](https://www.fachords.com/guitar-lessons/) | `FREEMIUM` | Step-by-step lessons map from beginner to advanced, plus interactive fretboard tools. |
| [Good Guitarist](https://goodguitarist.com/) | `FREE` | Easy chord tutorials, strumming patterns, song-based lessons. |
| [Riff Interactive](https://www.riffinteractive.com/) | `FREEMIUM` | Interactive software-based lessons with jam tracks, beginner to advanced. |
| [Online Guitar School Toolbox](https://onlineguitarschool.co.uk/toolbox/) | `FREE` | Hand-picked links to free practice tools, theory sites, and gear resources. |
